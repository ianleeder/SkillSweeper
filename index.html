Skillsweeper

Minesweeper, without the luck.

View on GitHub Download .zip Download .tar.gz

Purpose

I love playing MineSweeper, but hate the random chance that detracts from the skill of playing the game. I set out to make a new version of MineSweeper that would remove the element of chance, leaving just a game of skill: SkillSweeper.

Status

This is a pretty raw alpha version of the application.

Implemented

  • Basic detection of safe cells and certain mines
  • Basic game state
  • Autoplay mode

To-Do
